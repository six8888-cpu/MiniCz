# MiniCz - 梅花易数桌面宠物

一款融合中国传统易经文化的 Chrome 浏览器桌面宠物扩展。点击可爱的小宠物，根据当前时间起卦，获得64卦中的吉凶占卜结果。

## 功能特性

### 🎯 梅花易数占卜
- 基于传统梅花易数时间起卦算法
- 完整收录周易64卦
- 每卦配有吉凶等级（吉/中等/凶）和卦辞解读

### 🐾 桌面宠物动作

| 动作 | 文件 | 触发条件 |
|------|------|----------|
| 站立 | `站立.gif` | 默认待机状态 |
| 挥手 | `挥手.gif` | 点击宠物时的反馈动画 |
| 蹦跳 | `蹦跳.gif` | 占卜结果为"吉"时 |
| 蹲下 | `蹲下.gif` | 占卜结果为"凶"时 |
| 睡觉 | `睡觉.gif` | 30秒无操作自动进入 |
| 奔跑 | `奔跑.gif` | 鼠标快速移动时触发 |

## 安装方法

1. 下载本项目
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 安装后宠物会出现在网页右下角
2. 点击宠物进行占卜
3. 查看卦象结果和吉凶
4. 聆听语音反馈
5. 可拖拽宠物到任意位置

## 项目结构

```
├── manifest.json      # Chrome扩展配置
├── content.js         # 主逻辑（宠物创建、交互、动画）
├── divination.js      # 占卜核心（64卦数据、起卦算法）
├── content.css        # 样式文件
├── assets/            # 资源文件
│   ├── *.gif          # 宠物动画
│   └── *.mp3          # 语音文件
├── icons/             # 扩展图标
└── store_assets/      # 商店资源
```

## 隐私说明

本扩展仅使用 `storage` 权限存储宠物位置和首次点击状态，不收集任何个人信息。

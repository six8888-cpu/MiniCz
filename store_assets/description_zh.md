# MiniCz - 梅花易数桌面CZ

## 简介
一款融合中国传统易经文化的浏览器桌面CZ扩展。点击可爱的小CZ，根据当前时间起卦，获得64卦中的吉凶占卜结果。

## 主要功能

### 🎯 梅花易数占卜
- 基于传统梅花易数时间起卦算法
- 完整收录周易64卦
- 每卦配有吉凶等级和卦辞解读

### 🐾 可爱桌面CZ
- 6种动态GIF动画（站立、挥手、蹦跳、蹲下、睡觉、奔跑）
- 3D鼠标跟随视差效果
- 30秒无操作自动进入睡眠状态
- 鼠标快速移动触发奔跑动画

### 🔊 语音反馈系统
- 30+条趣味语音
- 根据吉凶结果播放不同语音
- 5%概率触发彩蛋语音

### 🎨 精美界面
- 深色主题卦象弹窗
- 动态吉凶颜色标识（红色-吉、橙色-中等、绿色-凶）
- 流畅的动画过渡效果

## 使用方法
1. 安装扩展后，CZ会出现在网页右下角
2. 点击CZ进行占卜
3. 查看卦象结果和吉凶
4. 聆听语音反馈

## 隐私说明
本扩展仅使用 storage 权限存储首次点击状态，不收集任何个人信息。
